# Fontes e referências — Autópsia GA36 / R36S clone

Este documento lista as fontes externas utilizadas como base técnica
para a autópsia de referência do console GA36 (clone do R36S).

Todo o crédito pela engenharia reversa original, dumps e imagens
pertence aos autores listados abaixo.

---

## Autópsia técnica principal

- **Repositório:**  
  R36S console clone — GA36-MB V1.0  
  https://github.com/phaseloop/R36S-console-clone---GA36-MB-V1.0-20250730

- **Autor:** phaseloop

- **Conteúdo disponibilizado:**
  - documentação técnica detalhada
  - dumps de partições
  - kernel funcional
  - imagem de recuperação (recovery image)

---

## Imagem de recuperação

- **Arquivo:** r36-a33-recovery-img.zip  
- **Hospedagem:** MediaFire  
- **Link:** citado no README do repositório da autópsia

⚠️ Observação:
Este projeto **não redistribui** a imagem de recuperação.
O link deve ser obtido diretamente no repositório original.

## Uso das fontes neste projeto

- As fontes listadas são utilizadas **exclusivamente para estudo e referência**
- Nenhum conteúdo é reivindicado como produção própria
- Nenhum binário de terceiros é redistribuído sem necessidade explícita

## Nota final

Este projeto existe graças ao trabalho prévio de engenharia reversa
realizado pela comunidade.

A documentação aqui presente busca:
- preservar conhecimento técnico
- evitar perda de informação
- facilitar estudos futuros sobre este hardware específico
